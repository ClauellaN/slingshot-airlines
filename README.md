## Overview
Slingshot Airlines is a user-friendly flight booking application that allows users to browse available flights, book seats, and manage their reservations seamlessly. Whether you need to check your reservation status or make adjustments, Slingshot Airlines provides a smooth and intuitive experience.

---

## Features
- **Flight Selection**: Users can select available flights based on their preferences (departure, arrival, date, etc.).
- **Seat Booking**: Book your seats easily on the chosen flights.
- **Reservation Management**: Check the details of your reservation anytime.

- **Payment Gateway**: Implementing payment integration using Square API to enable secure online payments.

## Technologies Used
- **Frontend**: React.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **API**: Square API (for future payment integration)
- **UI Styling**: CSS
